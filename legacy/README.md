# Legacy Configs

These files are preserved for reference, but they are no longer the default new-machine setup.

- `vimrc.vundle`: the old plugin-heavy Vim setup.
- `tmux.conf.powerline`: the old tmux powerline setup.
- `iterm-profile.json`: the old iTerm/Smyck terminal profile.

The active root `.vimrc` and `.tmux.conf` are intentionally small fallback configs.
